    📖 Mode belajar

    Untuk pemula: mulai dari satu huruf (a, b), lalu kata pendek. Getaran panjang = titik Braille, pendek = jeda.

    Tekan tombol di atas untuk dengar getaran (pelan). Ulangi sampai hafal polanya.

    Cara pakai

    Buka dari HP atau komputer. Kosongkan URL lalu Hubungkan. Setelah terhubung, ketik pesan dan Kirim — penerima merasakan getaran sesuai Braille.

    Selamat datang di Kaito Whale

    1. Hubungkan ke WiFi yang sama (HP & pengirim).
    2. Buka app ini di HP penerima, ketik URL server lalu Hubungkan.
    3. Ketik pesan dan Kirim — penerima merasakan getaran = Braille (panjang = titik, pendek = jeda).

    Gunakan Mode belajar untuk latihan pola getaran.
